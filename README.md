# AI 선생님 - 디자인 시안

AI 아바타 기반 온라인 학습 서비스 웹 디자인 시안입니다.

## 서비스 주요 기능
- 사용자가 원하는 AI 아바타 설정
- 아바타 모습·음성으로 온라인 수업 진행
- 마이크/채팅으로 실시간 질문
- AI가 즉시 음성 응답
- 수업 중 퀴즈 및 집중도 체크 (카메라 시선 추적)
- 집중 이탈 시 알림
- 학부모 리포트 제공

## 디자인 원칙
1. 전연령 사용 편의성 - 큰 버튼, 명확한 레이블
2. 인기 웹 디자인 패턴 - 카드 UI, 히어로 섹션, 사이드바
3. 저작권 침해 요소 없음 - 직접 제작 아이콘/일러스트
4. 12세 미만 친화적 - 둥근 모서리, 귀여운 아바타, 퀴즈 UI
5. 밝은 색상 위주 - 파스텔/비비드 톤 (우주 테마 제외)
